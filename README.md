FolderSize – Утилита для измерения размера папок на 7 языках
Быстрый и информативный инструмент для анализа занимаемого места на диске.
Показывает размер папки, рекурсивно обходит подпапки, сортирует, выделяет топ‑N самых больших элементов, исключает скрытые файлы и выводит данные в удобном формате.
Реализован на семи языках программирования с единым интерфейсом командной строки.

🚀 Возможности
Рекурсивный подсчёт – вычисляет полный размер всех файлов в папке и подпапках (опция -r).

Человекочитаемый вывод – размер отображается в байтах, КБ, МБ, ГБ, ТБ (опция -h).

Сортировка – подпапки сортируются по размеру (убывание) с опцией -s.

Топ‑N – показать только N самых больших элементов (опция -t N).

Исключение скрытых – пропускать файлы и папки, начинающиеся с точки (опция --exclude-hidden).

Глубина обхода – ограничить глубину рекурсии (опция -d N).

Детальный режим – показывает размер каждого файла и папки (опция -v).

Цветной вывод – наглядное представление результатов в терминале.

Поддержка больших каталогов – обрабатывает папки с миллионами файлов, не загружая всё в память.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [путь] [опции]
Основные опции
Опция	Описание
-r, --recursive	Рекурсивно обрабатывать подпапки (включено по умолчанию)
-h, --human-readable	Выводить размер в удобном формате (КБ, МБ, ГБ)
-s, --sort	Сортировать подпапки по размеру (убывание)
-t <N>, --top <N>	Показать только N самых больших элементов
-d <N>, --depth <N>	Максимальная глубина рекурсии (0 – только текущая папка)
--exclude-hidden	Не учитывать скрытые файлы и папки (начинающиеся с .)
-v, --verbose	Подробный вывод (показывать каждый файл/папку)
-h, --help	Справка
Примеры
bash
# Показать размер текущей папки (рекурсивно)
python folder_size.py .

# Показать размер папки /home/user в человекочитаемом виде с сортировкой
python folder_size.py /home/user -r -h -s

# Показать только 5 самых больших подпапок в /var/log
python folder_size.py /var/log -t 5 -r -s

# Обойти только 2 уровня вложенности и исключить скрытые
python folder_size.py ~/projects -d 2 --exclude-hidden

🧠 Логика работы
Обход файловой системы – рекурсивный или ограниченный по глубине обход директорий.

Подсчёт размера – суммирование размеров всех файлов (игнорируя ошибки доступа).

Сбор информации – для каждой папки сохраняется список подпапок с их размерами.

Сортировка – по размеру (убывание) при включённой опции -s.

Фильтрация – исключение скрытых элементов при необходимости.

Вывод – форматированный список с цветами и единицами измерения.

✨ Дополнительные фичи
Обработка ошибок доступа – программа продолжает работу, сообщая о невозможности прочитать некоторые папки.

Кеширование результатов – для повторных вызовов в рамках одной сессии.

Многопоточность – в некоторых реализациях ускорение за счёт параллельного обхода.
